## v1.0.2 界面与启动优化

### 改动内容

- **项目入口调整** — 设置中的“社区与源码”改为项目主页入口，移除 QQ 群和 Gitee，改为 fork 与原作者 GitHub 链接
- **帮助文档更新** — 帮助、教程、关于、更新说明统一改为本地优先版本表述，并同步更新多语言引导文案
- **启动速度优化** — Electron 启动日志改为异步写入，端口清理改为异步执行，并缩短多处固定等待时间

## v1.0.1 存档系统增强

### 改动内容

- **存档版本升至 v2** — 新增按作品 ID 索引章节（`perWorkChapters`），确保多作品数据完整导出
- **完全向后兼容** — v1 旧存档文件仍可正常导入
- **新增导出偏好设置** — 主题、语言、API 配置文件、视觉设置等偏好随存档一起导出/导入
- **修复存档数据丢失** — 导出/导入改用持久化层读写 IndexedDB，不再依赖 localStorage

## v1.0.9 稳定性与启动体验修复
### 改动内容

- **修复主窗口误跳转外部接口** - Electron 现在会拦截离开本地页面的主窗口导航，外部链接改为交给系统浏览器打开，避免主窗口直接显示接口 JSON 白页
- **默认语言改为稳定初始化** - 应用首次启动或本地未保存语言设置时默认使用简体中文，不再依赖首次欢迎弹窗才能正常进入
- **移除首次欢迎弹窗** - 精简首次进入流程，避免语言状态为空时额外插入启动步骤
- **修正主题切换文案** - 统一亮色/暗色切换按钮与提示文案，避免按钮当前状态和目标状态显示混淆
